post(newObject: T, options?: RequestOptions): Observable< T | any >

post() method is used to add a T object in database with a POST http request

If the endpoint respects the Genese standards, it should return the new object. In this case, the create() method will return this object formatted with the T type. In the other case, the create() method will simply return the request response.

create() method needs that your app respects the Genese standards :

* The endpoint must wait a POST method.

* The api path must respect the REST api standards for a POST method (example : http://my-path/book/id)

In this example, the create() method adds a new book with the NestJS backend.

Open your navigator console. In the tab "Network", you'll find the data received from POST request. In the tab "Console", you'll can compare these original data with the formatted objects sent as Observable by Genese. As you can see, Genese formatted the returned object with "Book" type.
